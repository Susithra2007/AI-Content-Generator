# AI Content Generator

A simple AI-powered content generator website that transforms raw notes into actionable tasks or structured content. Built for ET AI Hackathon 2026 using HTML, JavaScript, and Replit AI integration.

## Features

- Input raw notes in a text box.

- Generate tasks or structured content with one click.

- Displays results instantly below the input.

## Tech Stack

- HTML, CSS, JavaScript

- Replit AI integration (no API key required)
